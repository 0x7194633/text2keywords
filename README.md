# text2keywords
Trained T5 and T5-large model for creating keywords from text.
Supported languages:
| lang | keyT5 | keyT5-large | loss | epoch done|
|:----------------:|:----------------:|:-------:|:------:|:-------:|
| Russian | + | + | + | 2.91 | 6 |
| English | + | + | - | NaN | NaN |

The other languages are supported through a translator.
